# Hermes

Hermes is a learning-oriented **in-memory key-value store** written in **Go**.


Hermes focuses on clarity, correctness, and design tradeoffs rather than feature
parity or production readiness.

---

## Scope

- Single-node system
- In-memory data storage
- Standard library only
- Architecture evolves as requirements emerge

---

## Status

Early development. Structure and features will evolve over time.
